# 发布流程 Playbook

## ⚠️ 核心外脑（强制引用机制）

> **为什么采用动态引用而不是合并？**
> 因为 `发布规则速查.md` 会随着你的月度复盘不断进化。如果写死合并进本文件，会导致数据过时。因此采用**运行时强制动态加载绝对路径**的机制，确保 AI 每次生成的都是当月最新版本的经验！

在执行任何生成前，AI 必须使用工具强制读取以下文件的绝对路径（不可跳过）：

- 👉 读取底层架构：`/Users/moxuan/Desktop/Jemma工作台/创作/配置/平台规则/发布规则速查.md`

## 全平台文案（触发：录好了/全平台文案/全平台排期）

1. **装载大脑**：强制读取并完全理解 `/Users/moxuan/Desktop/Jemma工作台/创作/配置/平台规则/发布规则速查.md`（包含：防去重瀑布流时间表、动态彩蛋修补法则、标题CTR公式与禁忌）。
2. 生成各平台文案（参考 `视频/发布方案/` 下历史模板排版）

## 发完了（触发：发完了/发好了/上线了）

1. 问佳蔓哪些平台标题/时间/标签有改动
2. 记录实际发布数据 → 追加到 `publish-tracker.md`
3. 如有改动 → 同步更新 `配置/发布规则速查.md` 中的偏好
4. 下次“全平台文案”自动应用最新偏好

## 🔄 月度规则迭代协议（触发：月度复盘/更新平台规则）

> ⚠️ 核心系统逻辑：绝不盲从静态参数，保持规则库在算法演进中的生命力。

当佳蔓说“更新一下平台规则” 或 “做下月度复盘”时：
1. **全网爬虫搜证（AI 主动介入）**：AI 助手需主动使用 Search Web 搜索（如：“2026年4月 小红书 最新算法规则”、“TikTok 最新限流机制与字数限制”），获取各大内容平台当月的最新潜规则、限制字数和推荐时间变动。
2. **底层数据倒逼**：深度读取本月 `配置/platform-analytics.md` 中的真实播放转化与粉丝画像数据。
3. **合成与重写**：将【AI全网截获的最新平台算法基石】+【佳蔓本月的爆款偏好分析】+【平台物理改版与字数限制】，三管齐下，全量重写覆写 `配置/平台规则/发布规则速查.md`。确保下个月生成的每一行文案都踩在最硬核的数据和算法之巅。

## 长文发布与全网公域转私域矩阵（触发：发长文）
**视频首发阵列结束后，使用以下顺序打穿长尾流量池：**
1. **网站 (基座)**：打包 Markdown 配合自动编译直邮 Github 部署。
2. **视频号与公众号互炒**：视频号视频首发存活后，作为核心组件插入《微信特供.html》推文之中，公众号定时晚上错峰推送。
3. **Substack 轰炸**：携带已经发布的 YouTube 链接和长文 HTML，设定于次日09:00全量推送邮件。
4. **X / IG (社交放大器)**：Apple Reminders 触发，必须在视频与长文均已在线后，全网抛下引流大礼包。
5. **社群高净值转化**：将干货提炼、架构图、长文基座与底层视频全套链接，作为结案抛入「用AI发电」与「知识星球」。


## 全平台强制质量校验规格（防呆UI映射法则）

| 平台 | 校验排版与格式硬性要求 |
|------|--------|
| **YouTube** | 标题≤100字，描述前三行抛核心亮点，必备：时间戳、导流官网。标签【严禁带#号，只用逗号分隔】。必须错开国内发布，选定次日上午 08:30 (北美通勤) |
| **TikTok** | 根本没有单列的标题栏，必须将大标题、正文、标签 100% 融合在一个 4000 字符内的大框中。必须错峰发布。|
| **视频号** | 第一行到最后一行全为正文（含带 # 标签），但必须在代码块或文本最后单独拆出一行 6-16 字的短标题，专门用来填底部独立的极窄输入框。必须勾选原创声明。 |
| **B站** | 标题≤25字物理限制。文案里的标签严禁带 #，最后留一处纯净的带回车标签叠区。**必须在底部的防呆检查单中，单独为其撰写一段长度 ≤233 字符的“粉丝动态”，以强鼓动性话语求弹幕护体和一键三连。** |
| **小红书** | 标题≤20字物理限制，正文前两句设悬念。必提醒句末加话题并在后台加入合集与勾选原创。 |
| **抖音** | 标题≤30字，区分“工作标题”和“简介”，标签带#≤5个。发文助手必须勾选自主声明。 |
| **X/Instagram** | 极度精简图文引流大礼包：网站原文链接 + [YouTube 直达链接] + GitHub 开源仓库地址。 |
| **知识星球/群聊** | 高净值摘要（300字内）+ B平台/YouTube视频弹幕链接 + 微信公众号原文链接。 |
