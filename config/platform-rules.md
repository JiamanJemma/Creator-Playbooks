## 标题策略（数据验证）

**高 CTR 公式：**
| 平台 | 风格 | 示例结构 | CTR 数据支撑 |
|------|------|---------|-------------|
| YouTube | 短标题+具体问题 | "三种方法本地跑X" | 20.41% / 10.13% |
| B站 | 疑问句+反转对比 | "跟AI说句中文就能做特效？" | 收藏>点赞，好奇心驱动 |
| 小红书 | 🤯震惊emoji+悬念+竖线 | "🤯说句中文就能做特效｜Remotion入门" | 曝光775(上期7倍) |
| 抖音 | 数字对比+生活共鸣 | "1个月做6个App" | 封面CTR 21.43% |
| 视频号 | 冒号知性/金句 | "绕路才知近路：xxx" | 涨粉+9/天 |

**禁忌：**
- 标题罗列多个工具名（CTR 跌到 1-2%）
- 宽泛主题无具体卖点（"vibe coding作品" 0.93%）
- 超过 40 字（YouTube 尤其敏感）
- 各平台标题雷同 — 必须选不同角度

**标题差异化流程：**
- 读取各平台高CTR特征 + 粉丝画像 → 每平台选不同角度
- 常用角度：疑问式 / 痛点式 / 好奇式 / 震惊emoji / 冒号知性 / 最短最直
- 直接给定稿标题（基于数据已选好），不再给3选1

**佳蔓偏好：** 成果展示型（"一个人做完…"）、情绪转折型（"做到崩溃→换个思路"）

次日：X 09:00

**自动化排版硬性要求（AI必读）：**
1. **时间绝对化与分散**：全平台文案生成时，必须在**每个平台名称下面**（而不是堆在一起组成一个总表）写明具体的建议发布时间及其公历日期/星期（例如 `⏰ 建议发布时间：4月2日（周四） 19:00`），**严禁使用抽象的“明天/后天”一笔带过**。
2. **算法防内卷（48小时冷却）**：如果佳蔓在当天刚发过长视频，那么当天的产出请自动将建议首发日推迟到**48小时之后**，以防止 YouTube/B站 等平台的前作流量被吞并。周四是 Tech/AI 赛道极佳的发布引爆点。
3. **标签栏无井号/逗号化 (B站与YouTube)**：专门针对需要在后台填入特定输入框的 "标签" 区域（比如 B站和 YouTube），提取出来并紧跟在对应的标题下方。并且必须抹去 `#` 号，全部用半角英文逗号 `, ` 分开（例如：`AI黑客松, AIAgent, 一人公司`），方便一键粘贴自动拆分块。
4. **全平台表单防呆映射**：各平台（包括B站、小红书等）输出区块必须严格按照官方页面填表顺序，即：【标题】➡️【标签】➡️【视频简介/正文内容】➡️【定时发布时间】➡️【粉丝动态】。**绝对禁止把时间或总体标签堆叠在各平台的大标题上方，必须置于正文或单平台专用标签的最底部。**
5. **单框平台去结构化排版**：针对那些后台只有**单独一个文本框**的平台（如抖音、TikTok、视频号、X、IG），严格禁止在生成的文案里带上形如 `**标题：**`、`**正文：**` 或 `【标题】`、`【简介】` 的废话前缀标签。直接**开门见山地连成一大段纯文本**输出（标题文字+回车+正文文字+回车+井号标签），方便佳蔓直接无脑全框全选、一键粘贴。

## 关键短板与优化方向

- **抖音完播率 1.5%**（同类 13.7%，差 9 倍）→ 前2秒必须强视觉/悬念
- **2秒跳出率 43-46%** → 生死线在前2秒
- **公众号朋友圈是唯一有效渠道** → 每篇必转朋友圈
- **YouTube 展示量靠长尾** → 标题含搜索关键词
